EtherIP / L2TPv3 Server Detail Settings

EtherIP / L2TPv3 over IPsec compatible routers can connect to Virtual Hub on the VPN Server and establish Layer-2 (Ethernet) Bridging. For instance, Cisco routers, NEC IX series and IIJ SEIL routers are recommended as compatible VPN routers.



In order to accept EtherIP / L2TPv3 protocol, define appropriate EtherIP / L2TPv3 Client Settings to identify the client site of the router beforhand. An EtherIP / L2TPv3 Client Setting must have a corresponding IPsec Phase 1 ID.
